To run the automation tests in here you need to have the following installed in your machine
* NodeJS 10:  https://nodejs.org/en/download/current/
* JDK 1.8+

To run the tests, in the terminal from your local copy of the nss-todo-automation code run:
1. npm install
2. npm update
3. then run the test: npm test automation/wdio.conf.js
